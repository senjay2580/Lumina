# Lumina 提示词助手 - 浏览器插件

在任何网页上快速调用你的 AI 提示词库。

## 功能特性

- 🔍 **快速搜索** - 在任何网页上搜索你的提示词
- 📋 **一键复制** - 快速复制提示词到剪贴板
- ⏎ **直接插入** - 将提示词直接插入当前输入框
- 🏷️ **分类筛选** - 按分类快速筛选提示词
- ⌨️ **快捷键支持** - `Alt+Shift+L` 快速打开面板
- 🖱️ **可拖拽面板** - 面板位置可自由调整

## 安装方法

### Chrome / Edge / Brave

1. 下载 `browser-extension` 文件夹到本地
2. 打开浏览器，进入扩展程序页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Brave: `brave://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `browser-extension` 文件夹

## 使用方法

### 方式一：点击插件图标

1. 点击浏览器工具栏中的 Lumina 图标
2. 使用你在 Lumina 应用中的账号登录
3. 搜索或浏览提示词
4. 点击提示词卡片查看详情或复制

### 方式二：使用快捷键

1. 在任意网页按 `Alt+Shift+L`
2. 打开悬浮面板
3. 搜索并选择提示词
4. 按 `ESC` 关闭面板

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+Shift+L` | 打开/关闭提示词面板 |
| `ESC` | 关闭面板 |

## 快捷键不工作？

如果 `Alt+Shift+L` 快捷键无法打开面板，请按以下步骤排查：

### 1. 重新加载插件
- 进入 `chrome://extensions/`
- 找到「Lumina 提示词助手」
- 点击刷新按钮 🔄

### 2. 刷新网页
- 重新加载当前网页（Ctrl+R 或 F5）

### 3. 检查快捷键设置
- 进入 `chrome://extensions/shortcuts`
- 找到「Lumina 提示词助手」
- 确认「打开/关闭提示词面板」的快捷键是 `Alt+Shift+L`
- 如果显示冲突，可以手动设置其他快捷键

### 4. 检查控制台日志
- 在网页上按 F12 打开开发者工具
- 切换到 Console 标签
- 查看是否有 `[Lumina] Content script loaded` 的日志
- 按 `Alt+Shift+L`，查看是否有 `[Lumina] Alt+Shift+L detected` 的日志

### 5. 特殊页面限制
以下页面无法使用快捷键：
- Chrome 内置页面（chrome://、edge://）
- Chrome 网上应用店
- 其他扩展程序的页面

## 常见问题

### Q: 登录失败怎么办？

1. 确认使用的是 Lumina 应用中注册的账号
2. 检查网络连接
3. 尝试重新登录

### Q: 提示词无法插入到输入框？

插件会尝试将提示词插入到当前聚焦的输入框。如果：
- 当前没有聚焦的输入框，提示词会复制到剪贴板
- 某些网站可能有特殊的输入框实现，可能不兼容

### Q: 面板位置如何调整？

拖拽面板顶部的标题栏即可移动面板位置。

## 隐私说明

- 插件使用内置的 Supabase 配置连接数据库
- 登录凭据存储在浏览器本地存储中
- 不会收集或上传任何用户数据

- Manifest V3
- Chrome Extension API
- Supabase REST API
